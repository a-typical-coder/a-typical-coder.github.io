About

atypicalcoder is what now?

'atypicalcoder' is an easy to use platform empowering you to better navigate this digital age. We do this by building tools & software for you to use, and by providing learning resources teaching you how to build them yourself. All for free. Forever.

Why does atypical coder exist?

Struggled to start. Still struggling through the whole learning process. If you are here, you probably can relate. Want to share all the resources I find during my journey. I would be thrilled if it helps even one person to start coding some cool projects! And if not, then it's still a great resource for myself, as well as a collection of challenging and fun little coding projects that will help me improve my coding skills.

We are an open source driven platform

Everything we build is released as OPEN SOURCE.

All the tools and every piece of software. All the code from projects we display on YouTube / our website. Everything. So what does this mean for you? Well, it means you can use it free of charge. Even commercially. You can do with it whatever you want. Learn from it, modify it, use it in your daily life, mess around with the code, experiment. Just have fun with it :)
